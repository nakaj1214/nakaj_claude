# Wolfram Foundation Tool — 提供される能力

## Wolfram Language の主な能力領域

Wolfram が LLM に提供できる能力は大きく以下に分類される。

### 数学・計算

| 能力 | 例 |
|---|---|
| 数式処理（記号計算） | 微分・積分・因数分解・展開 |
| 数値計算 | 高精度浮動小数点演算 |
| 方程式求解 | 代数方程式・微分方程式・連立方程式 |
| 線形代数 | 行列演算・固有値・行列式 |
| 統計 | 確率分布・仮説検定・回帰分析 |
| 離散数学 | グラフ理論・組み合わせ論・整数論 |

### 科学・工学

| 能力 | 例 |
|---|---|
| 物理定数 | 光速・プランク定数・ボルツマン定数 など正確な値 |
| 化学 | 元素情報・化合物・反応式・分子量 |
| 天文学 | 天体データ・軌道計算 |
| 単位変換 | SI・CGS・US カスタム単位間の正確な変換 |

### 知識・データ

| 能力 | 例 |
|---|---|
| 地理 | 国・都市・人口・座標 |
| 言語 | 語源・文法・翻訳 |
| 金融 | 株価・為替・経済指標（リアルタイム対応） |
| 歴史 | 年表・人物・出来事 |

### データ処理・可視化

| 能力 | 例 |
|---|---|
| プロット生成 | 関数グラフ・散布図・ヒストグラム |
| データ解析 | CSVの統計処理・クラスタリング |
| 画像処理 | 変換・フィルタリング・認識 |
| 自然言語 | クエリの解釈・要約・分類 |

---

## LLM だけでは不正確になりやすい → Wolfram を使うべき場面

```
❌ LLM のみ（ハルシネーションリスク）
  「円周率の1000桁目は？」
  「H2SO4 の分子量は？」
  「1ドル = 何円？」（リアルタイム情報）

✅ Wolfram に委ねる（正確）
  Wolfram: Pi の1000桁 → 確定値を返す
  Wolfram: MolecularMass["H2SO4"] → 98.072 g/mol
  Wolfram: ExchangeRate["USD", "JPY"] → 現在値
```
