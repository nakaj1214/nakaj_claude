# ファイルで計画管理

永続Markdownファイルで計画と文脈を管理する。セッションが切れても作業を継続できる。

詳細な手順: [INSTRUCTIONS.md](INSTRUCTIONS.md)

## リソース

- [INSTRUCTIONS.md](INSTRUCTIONS.md) — 3ファイルパターン・テンプレート・ワークフロー・実践例の全内容
- [evaluations/evals.json](evaluations/evals.json) — テストケース（作成推奨）
