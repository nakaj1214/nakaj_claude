# OCR Project

## Tech Stack
- **Backend**: PHP 8.3 / Laravel 12
- **Frontend**: Vanilla JavaScript (no React)
- **Python**: gcp_OCR, offline_OCR
- **Database**: MySQL/SQLite

## Project Structure
- `app/` - Laravel application
- `public/js/` - JavaScript modules
- `resources/views/` - Blade templates
- `OCR_GUI/` - Main application
- `gcp_OCR/`, `offline_OCR/` - Python OCR scripts

## Conventions
- Use Japanese for comments and documentation
- Follow Laravel conventions for PHP
- Use ES6+ for JavaScript
