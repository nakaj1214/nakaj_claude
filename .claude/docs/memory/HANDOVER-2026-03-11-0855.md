## Issues & Solutions
- **Codex `codex exec` タイムアウト問題**: `codex exec --sandbox workspace-write` は `.codex/instructions.md` + `CLAUDE.md` を自動読み込みしてタイムアウト。→ `mcp__codex__codex` ツール（`base-instructions` 省略版）を使用することで解決
- **Delete/Backspace が最初のセルしか消えない（3回目）**: `onDocumentKeyDown` の `document.activeElement.tagName === 'INPUT'` チェックが常に真になる。→ 修正方針: テーブル内の input + 複数選択あり の場合は一括クリアを実行するよう条件変更
- **セル選択outline残留**: `isSelectionExcludedCell` に `no-select-col` を追加したことで No列クリックが `excel_range_select_copy.js` をバイパスし、既存セル選択がクリアされなくなった。→ 修正方針: `ExcelRangeSelectCopy` に `clearSelection()` を公開し `table_ui.js` の No列クリックハンドラから呼び出す
