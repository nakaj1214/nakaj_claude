## 学習事項

| 観察 | 原因推定 | 改善アクション |
|------|---------|---------------|
| `excel_range_select_copy.js` が3セッションで30回以上編集された | outline/box-shadow が input に隠れる問題を事前にDOMで確認せず実装を繰り返した | 実装前に DevTools で `::after` の z-index が input より高いか確認する手順を verify-before-fix に追加 |
| `daily_items` パネルの配置が4回変更された | Blade コンポーネントの親子関係を把握せず配置場所を決定した | 配置変更前に home.blade.php の flex コンテナ構造をGrepで確認する |
| `--zoom-height` デフォルト値が3ファイルに分散していた | CSS変数と JS の定数が独立して管理されていた | `zoom.js` に単一の定数 `DEFAULT_ROW_HEIGHT` を定義し、CSS初期値・datatable.js の両方がそれを参照する設計に変更する |
| settings.json の matcher 変更・差し戻しが2セッションで繰り返された | 変更後の実動作確認をしないまま次のセッションで再度同じ試みが行われた | settings.json の hook 変更後は必ず実動作確認コマンドをコメントに残す |
