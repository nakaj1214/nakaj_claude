## Skill 候補

| 繰り返しパターン | 自動化方法 | Skill 名 | トリガー条件 |
|----------------|-----------|----------|-------------|
| 特定キーワードを複数ファイルから横断削除・置換する | `grep` で対象ファイルを列挙 → 一括 sed 置換 → diff 確認 | `bulk-keyword-replace` | 「〜という記述を全ファイルから削除/置換して」と言われたとき |
| JS の同一アンチパターンを複数ファイルで修正する | `grep -r` で全箇所を特定してから一括修正 | （bulk-keyword-replace に統合可） | JS 修正時に同じパターンが他ファイルにも存在する場合 |

## 学習事項
