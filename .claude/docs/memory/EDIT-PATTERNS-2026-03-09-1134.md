## 学習事項

| 観察 | 原因推定 | 改善アクション |
|------|---------|---------------|
| `disabledMcpjsonServers` の値が3回変わった | サブエージェントにキー名調査を任せ、実際に Claude Code を再起動して確認するステップが抜けていた | MEMORY.md に記録済み。設定変更系は「次セッション冒頭で動作確認」をチェック項目に追加 |
| `PMA_HOSTS` の revert | docker公式ドキュメントを確認せず試した | phpmyadmin の環境変数変更前に公式ドキュメントを確認するステップを挟む |
| `settings.json` PreToolUse matcher の6分 revert | Hook の matcher 変更は、フック自体の動作（Bash コマンド承認）に即影響するため、変更→テスト→revert のサイクルが短い | matcher 変更は必ず「現在の Bash フローが壊れないか」をチェックしてから適用 |
| `excel_range_select_copy.js` に5回編集 | 仮想スクロール対応が複数の関数に影響し、段階的に追加する必要があった。事前に影響範囲を洗い出していれば回数を減らせた可能性 | 大きな機能追加前に影響する関数リストを作成してから着手する |
