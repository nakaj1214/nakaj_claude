## Hook 候補

| 検出パターン | 防止方法 | Hook 種別 | 実装案 |
|-------------|---------|-----------|--------|
| `console.log` を含む `.js` ファイルがステージに追加される | コミット前に検出して警告 | PreToolUse(Bash: git commit) | `git diff --cached` で `console.log` を grep し、あれば警告メッセージを出力 |
| `Playwright` という文字列が `.claude/` 配下のファイルに書き込まれる | このプロジェクトでは antigravity が正式手段であることを通知 | PostToolUse(Edit/Write) | 書き込んだファイルパスが `.claude/` かつ `Playwright` を含む場合に警告 |
