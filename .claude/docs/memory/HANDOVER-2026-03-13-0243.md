## Key Decisions

| Decision | Reason |
|----------|--------|
| `cd src` 禁止、常に絶対パスで操作 | src/ は embedded git repo で cwd が固定されるとフックがブロック |
| Codex は MCP 方式（`mcp__codex__codex`）を使用 | `codex exec` Bash 直接実行はタイムアウト問題があるため廃止 |
| `base-instructions` パラメータ必須 | なしだと `.codex/instructions.md` 等を自動読み込みしコンテキスト圧迫 |
| Bootstrap ホバーを app.css で `!important` 上書き | btn_list の独自スタイルが競合するため |
| インライン JS 禁止（例外: index_url/storage_scope） | `.claude/rules/laravel/no-inline-js.md` で明文化済み |
| `$panel.css('display', 'flex')` を使う（`.show()` ではなく） | jQuery `.show()` は display:flex を壊す |
