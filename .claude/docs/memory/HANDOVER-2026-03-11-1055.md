## Issues & Solutions
- **Codex exec タイムアウト**: `codex exec` bash 実行が20分以上かかりタイムアウト。`mcp__codex__codex` MCP ツールに切り替え（INSTRUCTIONS.md も更新済み）
- **Bootstrap ホバー CSS の `inherit` バグ**: 最初の実装で `background-color: inherit` を使ったが親要素からの継承になり意図しない動作。各バリアントの実際の色値を `!important` で指定して解決
- **`$panel.show()` の display:block 問題**: jQuery の `.show()` が `display: block` を設定し flex レイアウトを破壊。`$panel.css('display', 'flex')` に変更
- **Blade view キャッシュ**: コード変更後も画面に反映されない場合は `php artisan view:clear` でクリア
