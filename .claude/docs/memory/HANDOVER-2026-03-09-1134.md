## Gotchas & Notes
- **DB に対して `DROP DATABASE` / `DROP TABLE` は絶対に使わない**。カラム不足は `ALTER TABLE ADD COLUMN` で対応すること
- `init.sql` は古いスナップショットでカラムが不足している（`line`, `last_outbound`, `deleted_at`, `skip_save_modal` がない）。直接インポートすると後続マイグレーションが失敗する
- Slack承認フック（`slack_approval.py`）はデーモンが起動していないと機能しない。daemon が落ちていると全Bash実行が承認待ちになる
- `.claude/settings.json` の Slack フックは現在 **有効** 状態に戻している
- Serena MCP は `disabledMcpjsonServers` でサンプルタイムアウトが頻発するため信頼性低い
