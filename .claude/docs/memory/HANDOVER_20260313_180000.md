## 発生した問題と解決策

### Bash フック cwd 問題

**問題:** `cd src` で cwd が src/ に固定。src/ 内に .git があるため `git rev-parse --show-toplevel` が src/ を返し、フックが src/.claude/hooks/ を探して全 Bash コマンドがブロックされた。
**解決策:** セッション再起動で cwd がリセットされる。今後は `cd src` を使わず、常に絶対パスで操作する。
**注意点:** MEMORY.md に記録済み。src/ は embedded git repo であることを常に意識する。

### Codex レビュー収束

**問題:** Codex レビューが 1 回目で 5 件の blocking issues を検出。
**解決策:** 3 回の反復で全て修正し APPROVED を取得。主な修正: REQ-008 スコープ整合、Step 3 ロールバック方針（git stash → commit + revert）、完了条件の測定可能化。
