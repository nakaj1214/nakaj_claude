## Gotchas & Notes
- `DROP DATABASE` / `DROP TABLE` は絶対に使用しない。スキーマ変更は `ALTER TABLE` のみ
- `.stock_table input` は `width: 100%; height: 100%` で td 全体を覆うため、td の style は input に隠れる
- Codex 実行は `run_in_background: true` + `TaskOutput` で待機（リダイレクトなし、デフォルトモデル `gpt-5.4`）
- outline の問題は **input に `.css('box-shadow', ...)` を直接適用** する方針だが、input が `display: block` でない列（No列など）は td に適用が必要
