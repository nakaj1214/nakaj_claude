# スキル候補レポート — 2026-03-06 10:39

## 検出されたパターン

- **推測ベースの修正ループ**: 証拠なしにコードを読んだだけで原因を断定し、修正 → 失敗 → 再修正を繰り返すパターン
- **フロントエンドに偏ったルール**: 既存の `ui-fix-verification.md` が JS/CSS のみを対象としており、PHP・Python・DB レイヤーをカバーしていなかった
- **antigravity IDE との連携**: ブラウザ操作（スクリーンショット・DOM・コンソールログ・クリック）が可能なのに、活用フローが未整備
- **修正前の「ゲート」不在**: 「直して」と言われたら即コードを触り始め、原因確定フェーズがスキップされる
