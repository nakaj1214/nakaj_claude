## 学習事項

| 観察 | 原因推定 | 改善アクション |
|------|---------|---------------|
| `proposal.md` が1セッションで7回書き直された | 実装しながら要件が変わるため、create-plan の「計画を固めてから実装」原則が守られていない | `plan-rewrite-alert` フック（作成済み）を hooks に登録し、実際に有効化する |
| Delete/Backspace ハンドラが3セッションで3回書き直された | フロントエンド JS の自動テストがなく、ブラウザ手動確認に頼っているため退行に気づかない | `node --check` による構文チェックを PostToolUse フックに追加する（`excel_range_select_copy.js` 編集後） |
| settings.json の変更試行錯誤で2回 revert が発生 | テスト手順がなく、本番設定ファイルで直接試行している | `settings.json` 変更後に diff を表示し「次セッションで反映確認」を自動リマインドするフックを追加（MEMORY.md に「設定変更後は次セッションで反映確認すること」と既記載だが未自動化） |
| `review.md` が Codex レビュー結果 Write のたびに上書きされる | 前回のレビュー内容が残らず追跡できない | `review.md` 書き込み前に `docs/implement/review-history/` へ退避する仕組みを追加 |
