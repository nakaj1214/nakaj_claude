| 観察 | 原因推定 | 改善アクション |

| `proposal.md` が7回書き直された | 実装しながら要件が変わり、proposal が "実装メモ" に変質した | `create-proposal` 実行前にユーザーと受入条件を口頭確認し、fix-escalation の停止条件を proposal に明記する |
