## Issues & Solutions
- **Blade キャッシュ**: コード変更が反映されない → `php artisan view:clear` で解決
- **`$.show()` が flex を壊す**: `$panel.show()` → `$panel.css('display', 'flex')` に変更
- **合計金額が更新されない**: サジェスト `setFieldValue` が `change` イベントを発火しない → `controller.js` の `updateRow` 末尾で `ordered_number` の `change` イベントを明示的に発火
- **在庫 `flex: 1` 化後の表示サイズ変更バグ**: `height: 100%` に変えた影響で `zoom_level` 変更ロジックが意図通り動かなくなった（**未解決**）

- `order.blade.php` の `@if ($is_view && in_array($department_id, [0, 5]))` — 権限チェックあり

- 在庫画面の表示サイズ変更（`table_ui.js` の zoom_level 処理）は複雑。`flex: 1 / height: 100%` への変更後に動作確認が必要
