## 注意事項

- **`codex-review` の Codex 呼び出し**: `Task` ツールでサブエージェントを起動し、レビューチェックリストをプロンプトに埋め込む設計。`codex exec` CLI ではなく Claude サブエージェントで代替している
- **fnmatch の `**/` パターン**: `**/wp-config.php` はファイル名のみ（`wp-config.php`）にはマッチしない。テスト時はディレクトリプレフィックス付きのパスを使う
- **edit-tracker の機密マスキング**: `sk-[a-zA-Z0-9]{20,}` はハイフンを含まないので `sk-proj-xxx` 形式の OpenAI キーは先頭 `sk-` までしかマッチしない。意図的な仕様か要確認
