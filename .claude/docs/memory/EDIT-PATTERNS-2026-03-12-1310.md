## 学習事項

| 観察 | 原因推定 | 改善アクション |
|------|---------|---------------|
| `excel_range_select_copy.js` が 50+ 回編集された | input 要素が td を覆う構造を最初に把握せず実装に入った | CSS 環境を `verify-before-fix` で先に調査してから実装計画を立てる |
| `settings.json` の値が往復した | 設定変更後にブラウザ/Claude Code を再起動して動作確認していない | 設定変更は1ステップずつ確認を挟む（MEMORY.md に記録済み） |
| `plan.md` のレビューが 12 ループに達した | proposal の要件が大きすぎ・複数の独立した変更が混在していた | `propose-one` スキルで1要件ずつ進める。12要件一括は避ける |
| Blade ファイル上で HTML 構造を試行錯誤した | 実装前に CSS レイアウトの完成形を決めていなかった | レイアウト変更は先に CSS の疑似コードを `proposal.md` に明記してから Blade を変更する |
